# SpringBoot 쇼핑몰 프로젝트


## 개발 환경
* Java 11 (JDK 11.0.17)
* Server : Apache Tomcat 9.0
* Framework : Springboot, JPA
* Database: MySQL
* IDE : IntelliJ
* Build tool : Gradle

## 사용 기술
### [Back-End]
* JAVA 11
* SpringBoot
* Spring Data JPA
* MySQL

### [Front-End]
* Thymeleaf
* HTML, CSS, JavaScript
* jQuery
